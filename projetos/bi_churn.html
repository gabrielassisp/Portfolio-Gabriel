ChurnIQ Dashboard

Dashboard analítico construído do zero em JavaScript puro, sem frameworks nem bibliotecas externas de visualização. Lê e processa o dataset IBM Telco diretamente no browser via fetch() + parser CSV próprio, calcula todas as métricas em tempo real e renderiza gráficos interativos com Canvas API — com tooltips, dark mode e design responsivo.

Reflexões
O que este projeto me ensinou
⚙
JavaScript como pipeline de dados
Fazer o pipeline completo — leitura, parse, transformação, cálculo e visualização — em JS puro mostrou que o browser é um ambiente de análise de dados capaz e autossuficiente, sem precisar de Python, banco de dados ou servidor.
▦
Canvas API é mais poderosa do que parece
Gráficos de colunas agrupadas, donuts com hitbox trigonométrica e tooltips contextuais são plenamente realizáveis sem Chart.js ou D3. O controle total sobre o pixel é uma vantagem quando o design importa.
◈
Dados dinâmicos vs. valores fixos
A diferença entre um dashboard que apresenta dados e um que os processa é enorme para quem avalia um portfólio. Quando os insights mudam com o dataset, fica evidente que a análise é real — não uma apresentação estática.
